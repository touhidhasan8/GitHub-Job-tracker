What is the difference between map(), filter(), and forEach()?

Answer: 
In JavaScript, map(), filter(), and forEach() — all three are used to loop over arrays. But their purpose and return value are different.

map()->
It operates on each element of the array and returns a new array.

filter()->
Selects elements according to the condition and returns a new array.

forEach()->
Array loops do some work, but don't return anything.
